# Configuration for Oh-my-posh

## Prerequisites

- A [Nerd Font](https://www.nerdfonts.com/) installed in your system and enabled in your terminal.

This configuration works with zsh and [Oh My Zsh framework](https://ohmyz.sh).

1. Install [Oh-my-posh](https://ohmyposh.dev/docs).

2. Clone this folder or create a symbolic link into ~/config

3. Adding the following line at the end of file .zshrc

   > eval "$(oh-my-posh init zsh --config '~/.config/oh-my-posh/custom-atomic.omp.json')"

**Note:** You can use custom-atomic.omp.json or custom-amro.omp.json in this folder, or you can download a [theme](https://ohmyposh.dev/docs/themes)
for using in your terminal.
